# Memes Maker
## About
React.js responsive web application where users can create and post memes, also they can browse others memes and edit or delete them.
## Demo
![demo](https://github.com/hanan-tuwaiq/FinalProject/demo.gif)
## Technologies Used
### Front-end:
- Javascript
- React js
- Tailwind css
- Headless UI
### Back-end:
- Cloud Firestore API
- Axios
